🎵 Configure Feishin for Music Platform

⚠️ Important: Make sure your music platform backend is running on http://localhost:8000 before proceeding!

Step 1: Open Feishin

Launch the Feishin application on your computer.

Step 2: Go to Settings

Click on the settings icon and select "Servers" from the menu.

Step 3: Add New Server

Click the "Add Server" button and enter these details:

{ "id": "music-platform-backend", "name": "Music Platform Backend", "type": "music-platform", "url": "http://localhost:8000", "credential": "default-credential" }

If you haven't started the backend yet, run this command:

cd backend uvicorn app.main:app --reload --host 0.0.0.0 --port 8000
